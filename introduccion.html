Introduccion

Bienvenido a la Unidad numero 1 de Programando con DROPSY!
En esta unidad te mostraremos como utilizar DROPSY para crear tus proyectos!

Nuevo proyecto

Lo primero que debes aprender es a crear nuevos proyectos.
Crear un nuevo proyecto es muy facil, solo debes pulsar el boton (+):

Y luego pulsar en "Nuevo Proyecto":

Utilizando bloques

Los bloques nos permiten indicar las operaciones que queremos realizar
Para utilizar un bloque basta con seleccionar una categoria, por ejemplo Movimiento:

Al abrir una categoria se mostraran los bloques que pueden ser arrastrados al proyecto.

Arrastramos el bloque Mover como ejemplo:

Podemos combinar multiples bloques, los cuales seran ejecutados de arriba hacia abajo.
En este ejemplo se ejecutara primero el bloque rojo, segundo el violeta y ultimo el verde.

Para ejecutar nuestro proyecto simplemente pulsamos el boton Ejecutar:

Guardar proyecto

Para guardar nuestro proyecto pulsamos el boton Guardar:

Luego ingresamos el nombre para nuestro proyecto y oprimimos GUARDAR:

Cargar proyecto

Para cargar un proyecto guardado debemos pulsar el boton (+):

Y luego pulsar en "Cargar Proyecto":

Luego selecciona el proyecto que quieras cargar del listado:
