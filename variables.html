Unidad 4: Variables

Bienvenido

Bienvenido a la Unidad numero 4 de Programando con DROPSY!
En esta unidad aprenderas como usar variables en tus proyectos!

Creando Variables

Las variables son contenedores donde podes almacenar valores.
Luego, durante la ejecucion de tu proyecto, podes utilizar el valor contenido dentro de los contenedores.
Por ejemplo, podes tener una variable llamada cantidad que almacene la cantidad de veces que queres mover el robot.

Luego, podes utilizar esa variable en tu proyecto, en lugar de usar numeros:

Para lograr esto simplemente debemos abrir la categoria Variables

Y arrastramos el bloque "poner item"

Editando Variables

Para renombrar la variable pulsamos el nombre y luego seleccionamos "Renombrar Variable":

Ingresamos el nombre que queremos y pulsamos OK

Para editar el valor que queremos almacenar pulsamos el valor e ingresamos el nuevo valor.
